# 💊 Medi-Minder

A mobile-first, cloud-powered medication reminder app built with vanilla JavaScript and Firebase.

## ✨ Features

- 📝 Add, Edit, Delete medicines with multi-dose scheduling (Morning, Afternoon, Night)
- ☁️ Real-time cloud sync across devices using Firebase Realtime Database
- 🔔 In-app alarm with full-screen red alert + Vibration + Text-to-Speech
- 📱 Mobile-first responsive design
- 🚀 Push notifications ready (Firebase Cloud Messaging)

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Backend**: Firebase Realtime Database, Firebase Cloud Messaging
- **Deployment**: Vercel
- **Version Control**: Git & GitHub
